# 백년가게맵 계정 및 데이터 삭제 안내

- **앱 이름**: 백년가게맵 (Baeknyeon Store Map)
- **개발자**: 김건오
- **문의 및 삭제 요청 이메일**: gunoh.dev@gmail.com

이 페이지는 백년가게맵 이용자가 Google 또는 Apple 로그인으로 생성한 앱 계정과 관련 데이터의 삭제를 요청하는 방법을 안내합니다.

백년가게맵은 앱 내 더보기 > 내 정보에서 직접 계정 삭제 요청을 제출할 수 있습니다. 앱을 이미 삭제했거나 앱에 접근할 수 없는 경우, 또는 앱 내 절차가 어려운 경우에는 이 페이지의 이메일 요청 절차로도 삭제를 요청할 수 있습니다.

## 계정 삭제 요청 방법

계정 및 데이터 삭제는 다음 두 가지 방법 중 한 가지로 요청할 수 있습니다.

### 방법 1: 앱 내 계정 삭제 요청

1. 백년가게맵 앱 하단의 **더보기** 탭으로 이동합니다.
2. **내 정보** 화면을 엽니다.
3. **계정 삭제 요청**을 선택합니다.
4. 안내에 따라 재인증을 완료하면 요청이 운영자에게 전달됩니다.

앱 내 요청은 운영자에게 전달되어 검토 후 처리되며, 처리 완료 후 이메일로 회신합니다. 처리 원칙은 아래 "방법 2"와 동일하게 **영업일 기준 7일 이내**입니다.

### 방법 2: 이메일을 통한 요청

앱을 이미 삭제했거나 앱에 접근할 수 없는 경우, 또는 앱 내 절차가 어려운 경우 아래 절차에 따라 이메일을 보내 주세요.

1. `gunoh.dev@gmail.com`으로 이메일을 보냅니다.
2. 메일 제목을 **"백년가게맵 계정 삭제 요청"**으로 작성합니다.
3. 메일 본문에 다음 정보를 포함해 주세요.
   - 백년가게맵 로그인에 사용한 Google 또는 Apple 계정 이메일 주소
   - 삭제 요청 내용: 계정 및 관련 데이터 삭제 요청
   - 선택 사항: 요청 사유

본인 확인을 위해 가능하면 백년가게맵 로그인에 사용한 계정 이메일 주소에서 직접 요청해 주세요. 요청 이메일 주소와 앱 로그인 이메일 주소가 다른 경우, 본인 확인을 위해 추가 정보를 요청할 수 있습니다. Apple 로그인의 경우 이용자가 비공개 릴레이 이메일을 사용 중이라면 해당 릴레이 주소에서 요청해 주세요.

요청은 접수 후 **영업일 기준 7일 이내**에 처리하는 것을 원칙으로 하며, 처리 완료 후 이메일로 회신합니다. 본인 확인이 어렵거나 추가 확인이 필요한 경우 처리 기간이 더 소요될 수 있습니다.

## 삭제되는 데이터

계정 삭제 요청이 처리되면 다음 데이터가 삭제됩니다.

- **Firebase Authentication 계정 정보**
  - 백년가게맵 Google 또는 Apple 로그인으로 생성된 사용자 식별자(uid)
  - 로그인 이메일 주소(Apple 로그인의 경우 비공개 릴레이 이메일 포함 가능)
  - 표시 이름 등 인증 계정에 연결된 정보
  - Apple 로그인 계정의 경우, 처리 과정에서 Apple ID 인증 토큰의 폐지(revoke)를 함께 수행합니다.

- **즐겨찾기 백업 및 동기화 데이터**
  - Cloud Firestore `users/{uid}/snapshots/favorites`
  - 해당 계정에 연결된 즐겨찾기 목록, 즐겨찾기 분류, 정렬 순서, 저장 시각 등
  - 과거 버전에서 생성된 legacy 즐겨찾기 경로가 남아 있는 경우 해당 데이터도 함께 삭제

- **계정 식별자와 직접 연결된 서버 저장 데이터**
  - 백년가게맵 서버 또는 Firebase에 저장된 데이터 중 해당 uid와 직접 연결되어 계정 식별이 가능한 데이터

삭제가 완료되면 서버에 백업된 즐겨찾기 데이터는 복구할 수 없습니다.

## 별도 처리되거나 보관될 수 있는 데이터

다음 데이터는 계정 삭제와 별도로 처리되거나, 운영상 필요한 범위에서 삭제 또는 익명화될 수 있습니다.

- **정보 수정 제안 및 제보 데이터 (`store_reports`)**
  - 사용자가 제출한 가게 정보 수정 제안은 가게명, 주소, 수정 사유 등 가게 데이터 품질 개선에 필요한 운영 기록입니다.
  - 계정 삭제 처리 시 제보에 연결된 사용자 식별자(`userId`)는 삭제하거나 익명화합니다.
  - 사용자가 제보 메모에 직접 입력한 개인정보가 포함되어 있고 해당 내용의 삭제를 원하는 경우, 삭제 요청 메일에 함께 알려 주세요. 확인 후 해당 개인정보를 삭제하거나 식별할 수 없도록 처리합니다.

- **가게별 익명 통계 (`store_stats`)**
  - 가게별 즐겨찾기 누적 수치 등 통계 데이터는 특정 사용자를 식별할 수 없는 집계 데이터입니다.
  - 계정 삭제 후에도 개인 식별 정보 없이 보관될 수 있습니다.

- **앱 진단 및 분석 데이터**
  - Firebase Analytics, Firebase Crashlytics 등 외부 서비스가 처리하는 앱 이용 통계, 오류, 비정상 종료 정보는 백년가게맵 계정과 직접 연결하지 않는 진단 목적 데이터입니다.
  - 해당 데이터는 각 서비스 제공자의 보관 및 삭제 정책에 따라 처리됩니다.

- **문의 이메일**
  - 삭제 요청이나 문의 대응을 위해 사용자가 이메일로 제공한 내용은 요청 처리 및 분쟁 대응을 위해 필요한 기간 동안 보관될 수 있습니다.

## 기기 내 로컬 데이터 삭제

계정 삭제 요청은 서버에 저장된 계정 및 관련 데이터 삭제를 위한 절차입니다.

이용자의 기기 내부에 저장된 로컬 데이터는 다음 방법으로 직접 삭제할 수 있습니다.

- 기기에서 백년가게맵 앱 삭제
- Android 설정에서 백년가게맵 앱 데이터 삭제

앱을 삭제하거나 앱 데이터를 삭제하면 기기 내부에 저장된 즐겨찾기, 설정, 최근 선택 지역 등 로컬 데이터가 삭제될 수 있습니다.

## 삭제되지 않는 항목

다음 항목은 계정 삭제 요청만으로 삭제되지 않습니다.

- 이용자의 Google 계정 또는 Apple ID 자체
- Naver Map, KakaoMap 등 외부 앱 또는 외부 웹사이트에서 사용자가 직접 이용한 기록
- 법령 준수, 보안, 부정 이용 방지, 분쟁 대응 등 정당한 사유로 일정 기간 보관이 필요한 정보
- 개인을 식별할 수 없는 집계 통계

백년가게맵 계정 삭제는 Google 계정 또는 Apple ID 자체의 삭제가 아니며, Google 계정과 Apple ID는 각 서비스의 계정 설정에서 별도로 관리해야 합니다. 앱 내 로그아웃만으로는 계정 또는 서버 저장 데이터 삭제 요청이 처리되지 않습니다.
